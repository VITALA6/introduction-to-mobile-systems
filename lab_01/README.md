# Lab 01 – Random Number Generation and Distribution Simulation

## Co zostało zrealizowane
Zaimplementowano własny generator liczb pseudolosowych (LCG – Linear Congruential Generator) oraz funkcje generujące próbki z rozkładu Poissona i rozkładu normalnego (metoda Box-Mullera). Wyniki wizualizowane są jako histogramy przy użyciu matplotlib.

## Uruchomienie
Wymagana instalacja zależności:
```bash
uv venv && source .venv/bin/activate
uv pip install -r requirements.txt
python lab1.py
```
